# OpenBrain Branding Guidelines

These guidelines define how to refer to the project and how to brand forks responsibly.

See also: `TRADEMARK.md`.

## Official name styling
- Use `OpenBrain` (capital O and B).

## How to reference forks
- Preferred wording:
  - "Fork of OpenBrain"
  - "Based on OpenBrain"
- Do not present a fork as the official OpenBrain project.

## "Compatible with OpenBrain" wording
- Allowed when accurate.
- Must not imply endorsement, certification, or formal partnership.
- Include clear naming separation from the OpenBrain project.

## Logo usage
- If no official logo pack is published, assume no logo usage rights are granted.
- If logo assets are published in the future, use them only under the terms in `TRADEMARK.md`.
