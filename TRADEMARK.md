# OpenBrain Trademark Policy

OpenBrain is open source software, and forks are welcome. This policy explains how the OpenBrain name and branding may be used so users are not confused about what is official.

## Covered marks
- `OpenBrain` word mark.
- Official OpenBrain logos and brand graphics (if and when published).

## What is allowed
- Forking, modifying, and redistributing the code under the repository software license.
- Truthfully describing lineage, for example:
  - "Forked from OpenBrain"
  - "Based on OpenBrain"
- Nominative fair use, such as "Compatible with OpenBrain", when accurate and not misleading.

## What is not allowed
- Naming a forked product or hosted service `OpenBrain`.
- Using OpenBrain logos/brand assets in a fork or derivative product.
- Implying endorsement, partnership, certification, or official status when none exists.
- Using confusingly similar branding, including names like `OpenBrain Pro`, `OpenBrain Enterprise`, or similar variants, without permission.

## Fork compliance checklist
- Rename your forked product/service to a distinct name.
- Remove OpenBrain logos/brand assets from your forked distribution.
- Keep an attribution statement such as "Forked from OpenBrain" or "Based on OpenBrain".
- Avoid any wording that suggests official OpenBrain endorsement.

## Permission requests
If you need explicit trademark permission (for example, co-branding, reseller packaging, or promotional use), contact maintainers via GitHub issues in this repository.
